## 📝 劇本摘要

### 故事大綱
```
深夜 11 點，美術館「月光之淚」鑽石被盜。
私家偵探林明受命調查。
三名嫌疑人：保安隊長、策展人、捐贈者。
通過勘察現場、詢問證人，發現保安隊長證詞矛盾。
最終指認正確兇手，達成真相結局。
```

### 關鍵矛盾
```
張偉說：「監控從 8 點開始壞了」
但張偉：「9 點才開始值班」
矛盾：他怎麼知道監控確切故障時間？
```

### 真相
```
張偉利用職務之便複製了展櫃鑰匙。
從內部敲碎玻璃，延遲警報觸發。
本想製造假案，沒想到警報器故障提前響了。
因欠下賭債，想監守自盜。
```
