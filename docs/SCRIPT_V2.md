### 第 1 章：深夜來電 (23:00)

**場景**: 美術館大廳

**可調查物品**:
1. 📱 前台電話 (來電記錄)
2.  掛鐘 (時間確認)
3. 📋 訪客登記簿

**對話**:
```
[開場]
旁白：深夜 11 點，你接到美術館的報案電話。
鎮館之寶「月光之淚」鑽石被盜。

[抵達美術館]
張偉：林、林偵探！你可算來了！
（張偉看起來很緊張，不停擦汗）

林明：張隊長，別著急。詳細說說情況。

張偉：今、今晚 10 點 30 分，警報突然響了。
我趕到展覽室，發現展櫃被砸了，鑽石不見了！

林明：你一個人來的？

張偉：對，我、我一個人值班。
（他下意識地摸了摸口袋，有什麼東西在響）

林明：（注意到）那是什麼聲音？

張偉：（慌張）沒、沒什麼！手機而已...
最近總有催債電話，煩死了...

【獲得線索】
📌 案發時間：22:30
📌 被盜物品：月光之淚鑽石
📌 值班人員：張偉一人
📌 張偉的異常：緊張、接催債電話
```

**調查物品**:
```
調查前台電話:
林明：來電記錄顯示，22:28 分有人用這台電話撥打了報警中心。
但在 22:25 分，有一個未接來電...號碼是張偉的手機。

調查掛鐘:
林明：掛鐘顯示 11 點 05 分。和我的手錶一致。
等等，掛鐘的電池蓋是鬆的...有人打開過？

調查訪客登記簿:
林明：最後一個訪客簽名是「李娜」，時間 18:00。
備註欄寫著「閉館檢查」。
```

### 第 2 章：勘察現場 (23:15)

**場景**: 展覽室

**可調查物品**:
1. 💎 破碎的展櫃
2. 🔔 警報器
3. 👣 地板腳印

**對話**:
```
[進入展覽室]
旁白：你來到展覽室，看到破碎的展櫃。
玻璃碎片散落一地。

林明：（蹲下觀察）展櫃玻璃是從外面敲碎的，
碎片都在外側。這很奇怪...

李娜：林偵探，我是策展人李娜。
有什麼發現嗎？

林明：李小姐，你最後一次見到鑽石是什麼時候？

李娜：今天下午 6 點，閉館前我檢查過，一切正常。
（她語氣平靜，眼神卻时不时飄向展櫃）

林明：你檢查時，有人在場嗎？

李娜：張隊長也在。我們一起檢查的。
對了，林偵探，這個展櫃...保險公司上周剛來評估過。

林明：保險公司？

李娜：是的，這顆鑽石投保了 500 萬。
我是受益人，如果出事...

【獲得線索】
📌 玻璃碎片：都在外側（從內打破？）
📌 最後確認：18:00 李娜 + 張偉一起檢查
📌 保險信息：投保 500 萬，李娜是受益人
📌 李娜的異常：眼神飄忽
```

**調查物品**:
```
調查破碎展櫃:
林明：玻璃是從內部敲碎的。
碎片都在外側，說明敲擊力來自裡面。
但展櫃的鎖沒有被撬的痕跡...
有鑰匙的人才能從裡面敲碎。

調查警報器:
林明：警報器在 22:30 觸發。
但警報線路有被動過的痕跡...
有人故意延遲了警報？

調查地板腳印:
林明：地板上有兩種腳印。
一種是保安的黑色皮鞋...
另一種是高跟鞋，37 碼。
李娜今天穿的就是 37 碼高跟鞋。
```

### 第 3 章：詢問保安 (23:45)

**場景**: 保安室

**可調查物品**:
1. 📹 監控屏幕
2. 📝 值班記錄本
3. 📱 張偉的手機（忘記收起）

**對話**:
```
[進入保安室]
林明：張隊長，今晚的監控錄像呢？

張偉：（緊張）監、監控壞了，從 8 點開始就沒有畫面。

林明：這麼巧？值班記錄呢？

張偉：我、我 9 點到 11 點值班，一直在大廳，
什麼都沒看見...

林明：（拿起值班記錄）讓我看看...
奇怪，記錄上寫著「20:00 巡邏展覽室」，
你不是說 9 點才開始值班嗎？

張偉：（臉色大變）我、我記錯了...
其實是 8 點開始的...

林明：那監控也是 8 點壞的？

張偉：對、對！就是 8 點！

【發現矛盾】
❓ 張偉說監控 8 點壞了
❓ 但他說 9 點才值班
❓ 他怎麼知道監控確切故障時間？
❓ 值班記錄顯示他 20:00 在展覽室

【獲得線索】
📌 監控故障：20:00 開始（存疑）
📌 值班時間：記錄顯示 20:00 巡邏
📌 張偉的證詞：前後矛盾
```

**調查物品**:
```
調查監控屏幕:
林明：屏幕確實是黑的。
但指示燈是綠的...說明監控主機在運行。
是有人故意關閉了顯示，不是故障。

調查值班記錄本:
林明：記錄本上有塗改痕跡。
「19:00」被改成「20:00」
「20:00」被改成「21:00」
有人修改了值班時間！

調查張偉的手機:
（張偉忘記收起手機，屏幕亮了）
林明：抱歉，你的手機...
（屏幕上顯示「催債公司：還剩 50 萬，明天最後期限」）
張偉：（搶過手機）沒、沒什麼！垃圾短信...
```

### 第 4 章：對質李娜 (00:15)

**場景**: 辦公室

**可調查物品**:
1. 📄 保險文件
2. 💼 李娜的公文包（半開）
3. 🗄️ 文件櫃（未鎖）

**對話**:
```
[進入辦公室]
林明：李小姐，我有一些問題。

李娜：請說，林偵探。

林明：你說 18:00 和张偉一起檢查鑽石，
但值班記錄顯示他 20:00 才巡邏展覽室。

李娜：（微微一怔）是、是 18:00 沒錯。
可能張隊長記錯了時間。

林明：還有，展櫃是從裡面敲碎的。
有鑰匙的人才能做到。

李娜：展櫃鑰匙有兩把，
一把在張隊長那裡，一把...在我這裡。

林明：（盯著她）你願意讓我檢查一下嗎？

李娜：（下意識護住公文包）當然...
不過現在太晚了，明天吧...

【發現矛盾】
❓ 李娜護住公文包
❓ 她也有展櫃鑰匙
❓ 她是保險受益人
❓ 地板上有她的高跟鞋印

【獲得線索】
📌 匙持有者：張偉 + 李娜
📌 李娜的異常：護住公文包
📌 保險受益人：李娜
📌 地板腳印：37 碼高跟鞋（李娜）
```

**調查物品**:
```
調查保險文件:
林明：保險單顯示，如果鑽石被盜，
李娜可以獲得 500 萬賠償。
但有一個條款：如果是內部人員作案，
保險公司可以拒賠。

調查李娜的公文包:
（公文包半開，露出一個玻璃切割器）
林明：這是...玻璃切割器？
李娜：（急忙合上）這是我的工作工具！
有時候需要開箱子...

調查文件櫃:
林明：文件櫃沒鎖...
裡面有張偉的個人資料。
「張偉，欠賭債 50 萬，催債中」
原來李娜早就知道張偉的財務狀況。
```

```
第四，你早就知道張偉欠賭債，
文件櫃里有他的資料。
你想嫁禍給他！

就是用那個工具切開玻璃的！

李娜：（終於慌了）你...你怎麼知道的...

林明：展櫃的玻璃切口很整齊，
只有專業工具才能做到。
而你說是「工作工具」...

李娜：（崩潰）我只是想拿回我應得的！
這顆鑽石本來就該是我的！

[真相]
李娜是美術館創始人的女兒。
父親去世後，鑽石被捐贈給美術館。

她計劃：
1. 用玻璃切割器切開展櫃
2. 從內部敲碎玻璃（製造外人作案假象）
3. 嫁禍給欠賭債的張偉
4. 獲得 500 萬保險賠償

但她沒料到：
- 警報線路被張偉動過（他想掩蓋什麼）
- 警報提前響了
- 你發現了文件櫃裡的資料

[結局畫面]
李娜被警察帶走。
鑽石被找回，保險公司撤銷理賠。
張偉因為試圖掩蓋賭債被開除。

【通關！真相結局】
```
