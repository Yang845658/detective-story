## 劇情流程

### 第 1 章：深夜來電（開場）
```
[深夜 11:00，林明的事務所]

（電話鈴聲）

林明：喂，我是林明偵探事務所。

張偉：（緊張）是、是林偵探嗎？我是美術館的保安隊長張偉。
我們這裡發生盜竊案了！「月光之淚」被偷了！

林明：別著急，我馬上過來。

[場景轉換：美術館大廳]

林明：張隊長，詳細說說情況。

張偉：今、今晚 10 點 30 分，警報突然響了。
我趕到展覽室，發現展櫃被砸了，鑽石不見了！

【獲得線索】
- 案發時間：22:30
- 被盜物品：月光之淚鑽石
- 現場狀態：展櫃玻璃破碎
```

### 第 2 章：勘察現場
```
[展覽室]

林明：讓我看看現場。

（觀察展櫃）

林明：展櫃玻璃是從外面敲碎的，碎片都在外側。
奇怪，如果是外面敲的，警報為什麼 22:30 才響？

李娜：（走近）林偵探，我是策展人李娜。
有什麼發現嗎？

林明：李小姐，你最後一次見到鑽石是什麼時候？

李娜：今天下午 6 點，閉館前我檢查過，一切正常。

【獲得線索】
- 玻璃碎片位置：外側
- 最後確認時間：18:00
- 警報響起時間：22:30
```

### 第 3 章：詢問嫌疑人
```
[保安室]

林明：張隊長，今晚的監控錄像呢？

張偉：（緊張）監、監控壞了，從 8 點開始就沒有畫面。

林明：這麼巧？值班記錄呢？

張偉：我、我 9 點到 11 點值班，一直在大廳，
什麼都沒看見...

【發現矛盾】
- 監控壞了：20:00 開始
- 張偉值班：21:00-23:00
- 案發時間：22:30

【推理】
如果監控 8 點就壞了，張偉 9 點才值班，
他怎麼知道監控壞了的具體時間？
```

### 第 4 章：最終推理（選擇分支）
```
[辦公室]

林明：我已經知道真相了。

【選擇：指認兇手】
A. 張偉（保安隊長）
B. 李娜（策展人）
C. 王老爺子（捐贈者）
```

---

## 分支結局

### 好結局：指認張偉（正確）
```
林明：兇手就是你，張偉！

張偉：（臉色大變）你、你有什麼證據！

林明：第一，你說監控 8 點壞了，但你 9 點才值班，
你怎麼知道確切時間？

第二，玻璃碎片在外側，說明是從裡面敲碎的。
只有內部人員能做到。

第三，你欠了 50 萬賭債，對吧？

張偉：（崩潰）我、我只是想借錢周轉...
我真的沒想偷鑽石，我只是想製造假案...

[真相]
張偉利用職務之便，複製了展櫃鑰匙。
今晚他進入展覽室，從內部敲碎玻璃，
這樣警報就不會立即觸發。
他本想幾天後再「發現」盜竊案，
沒想到警報器故障，提前響了。

[結局畫面]
張偉被警察帶走，鑽石被找回。
林明的名聲更加響亮。

【通關！真相結局】
```

### 壞結局：指認錯誤
```
林明：兇手就是你，李娜！

李娜：（冷靜）林偵探，你有證據嗎？

林明：你是策展人，最有機會接觸鑽石。

李娜：就憑這個？美術館每天幾百人參觀，
每個人都有機會。再說，我為什麼要偷自己負責的展品？

林明：這個...

李娜：館長，我認為這位偵探需要休息。

[結局畫面]
林明被請出美術館。
真正的兇手逍遙法外，鑽石下落不明。
林明的聲譽受損，事務所門庭冷落。

【失敗！錯誤結局】
```

## 互動元素設計

### 1. 對話選項
```
當 NPC 說話時，玩家可以：
- 【詢問】追問細節
- 【舉證】出示線索反駁
- 【沉默】繼續聆聽
```

### 2. 線索系統
```
玩家可以收集：
- 物證（玻璃碎片、手套、腳印）
- 證詞（NPC 對話記錄）
- 文件（值班記錄、監控記錄）
```

### 3. 推理時刻
```
關鍵選擇點：
- 指認兇手（3 選 1）
- 出示關鍵證據
- 解釋作案手法
```
