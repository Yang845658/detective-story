# 🔍 午夜美術館 - 偵探推理互動小說

> **版本**: v1.0.0  
> **類型**: 偵探推理 / 互動小說  
> **預計時長**: 30 分鐘

---

## 📖 故事簡介

深夜 11 點，你接到美術館的報案電話。

鎮館之寶「月光之淚」鑽石被盜。

三名嫌疑人，一個真相。

你能找出真正的兇手嗎？

---

## 🎮 遊戲特點

### 核心玩法
- 🔍 **勘察現場** - 收集線索和證據
- 💬 **詢問嫌疑人** - 發現證詞矛盾
- 🧠 **推理分析** - 找出真正兇手
- ✅ **多重結局** - 你的選擇決定結局

### 技術特點
- 📱 **響應式設計** - 支持桌面和移動端
- 🎨 **精美 UI** - 漸變背景和動畫效果
- ⚡ **快速加載** - 純前端實現，無需服務器
- 🎯 **互動體驗** - 點擊對話、選擇分支

部署到 GitHub Pages、Vercel 或 Netlify。

## 📋 遊戲流程

### 第 1 章：深夜來電
- 接到報案電話
- 抵達美術館
- 會見保安隊長張偉

### 第 2 章：勘察現場
- 檢查破碎展櫃
- 會見策展人李娜
- 收集現場線索

### 第 3 章：詢問嫌疑人
- 詢問保安隊長
- 發現證詞矛盾
- 收集關鍵證據

### 第 4 章：最終推理
- 指認兇手（3 選 1）
- 真相結局 / 錯誤結局

---

## 🎯 結局指南

### ✅ 好結局：真相
**條件**: 指認張偉是兇手

**關鍵線索**:
- 值班記錄（張偉 9 點才值班）
- 監控記錄（監控 8 點壞了）
- 矛盾點：張偉怎麼知道監控確切故障時間？

### ❌ 壞結局：錯誤
**條件**: 指認李娜或王老爺子

**結果**: 真兇逍遙法外，聲譽受損

## 🎮 操作說明

### 桌面端
- **點擊對話框** - 繼續對話
- **點擊選項** - 做出選擇
- **查看線索本** - 回顧收集的证据

### 移動端
- **觸摸對話框** - 繼續對話
- **觸摸選項** - 做出選擇
- **滑動查看** - 線索和筆記

## 💡 遊戲提示

### 成功要訣
1. 📝 **仔細閱讀** - 每個細節都可能是關鍵
2. 🔍 **觀察矛盾** - 嫌疑人的證詞可能有漏洞
3. 🧠 **邏輯推理** - 基於證據做出判斷
4. ⏰ **注意時間** - 時間線是破案關鍵

### 常見錯誤
- ❌ 沒有收集足夠線索就指認
- ❌ 被嫌疑人的外表迷惑
- ❌ 忽略時間線矛盾
